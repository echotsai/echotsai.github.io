Shell的定义

维基给出的解释：
In computing, a shell is a user interface for access to an operating system’s services. In general, operating system shells use either a command-line interface (CLI) or graphical user interface (GUI), depending on a computer’s role and particular operation. It is named a shell because it is a layer around the operating system kernel.

在计算中，shell是用于访问操作系统服务的用户界面。一般来说，操作系统shell使用命令行界面（CLI）或图形用户界面（GUI），具体取决于计算机的角色和特定操作。它被命名为一个shell，因为它是操作系统内核周围的一层。)

Shell的作者

同样来源于维基，点击可查看原文：
Stephen Richard “Steve” Bourne (born 7 January 1944) is a computer scientist, originally from the United Kingdom and based in the United States for most of his career. He is well-known as the author of the Bourne shell (sh), which is the foundation for the standard command line interfaces to Unix.[1]

史蒂芬理查德“史蒂夫”布恩（生于1944年月7日）是一位计算机科学家，最初来自英国，在美国大部分职业生涯。他作为Bourne shell（sh）的作者而闻名，它是与Unix的标准命令行界面的基础

Markdown 语法练习

May 29, 2017
从零学编程

We believe that writing is about content, about what you want to say – not about fancy formatting.

我们坚信写作写的是内容，所思所想，而不是花样格式。
— Ulysses for Mac

“Markdown 作为一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。” [引]

最好的练习就是实践。为了更快熟悉Markdown的语法，我便以其语法说明文档为范本，做以下的进一步练习。

为了方便识别记忆，空格都用了下划线_表示。

标题

一级标题 –> #
二级标题 –> ##
三级标题 –> ###
四级标题 –> ####
五级标题 –> #####
六级标题 –> ######

列表

分为有序列表和无序列表。
-有序列表：
直接在文字前加1. 2. 3. 符号
要和文字之间加上一个字符的空格

-无序列表：
直接在文字前加+ - *符合
同样的，要和文字之间加上一个字符的空格

引用

为了突出该部分内容是引用而来的，可以在段落或句子的文字前面加>
其后要和文字之间加上一个字符的空格

图片与链接

当需要插入图片时，输入 ! + [要插入图片的内容] + (链接地址)

当需要插入链接时，输入[要插入链接的内容] + (链接地址)

粗体与斜体

如果要做斜体格式，输入*斜体内容*
如果要做粗体格式，输入**粗体内容 **

代码框 (block code)

如果需要创建1个代码区块：

  1. 可以用<pre> 和 <code> 标签来把代码区块包起来，比如：
    <pre><code>hello</code></pre>
    hello
    2. 或是在文字前缩进4个空格(空4个格)或是1个制表符(敲一下“tab”键)，然后上方空一行就可以了
  2. 最简单的是直接用3个” ` “前后包住代码即可

如果你不需要这么大面积的代码，只是想引用1-2个代码内容(行内代码 - inline code)，直接用” ` “前后包住代码即可

分割线

如果需要插入分割线，在一行中输入三个以上的连字符：* - _ 即可生成下划线

反斜杠

反斜杠可以帮助你获得某些字符本来的符号功能，
比如：* + 斜体内容 + * 即是 斜体内容
可是，现在我只想要用*作为一个强调符号！不要斜体！

怎么办？

例如：你只需要在星号的前面加上反斜杠：
\``*``斜体内容``\``* 即是 *literal asterisks*

除了*还有这些符号也同样适用：

\   反斜线
`   反引号
*   星号
_   底线
{}  花括号
[]  方括号
()  括弧
 #   井字号
+   加号
-   减号
.   英文句点
!   惊叹号hello
